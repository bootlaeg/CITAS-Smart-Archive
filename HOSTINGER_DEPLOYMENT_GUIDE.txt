PROFILE PICTURE FEATURE - HOSTINGER DEPLOYMENT GUIDE
====================================================

CURRENT SETUP (RECOMMENDED):
- Profile pictures stored as files in: uploads/profile_pictures/
- File paths stored in database
- Works perfectly on Hostinger and all shared hosting

HOSTINGER DEPLOYMENT STEPS:
==========================

1. UPLOAD YOUR CODE
   - Use Hostinger File Manager or FTP
   - Upload all files to public_html/

2. CREATE UPLOADS FOLDER (Important!)
   Using Hostinger File Manager:
   ✓ Go to File Manager
   ✓ Navigate to your public_html folder
   ✓ Create new folder: uploads
   ✓ Inside uploads, create: profile_pictures
   ✓ Right-click profile_pictures → Permissions → Set to 755

   OR via Terminal (if SSH access):
   mkdir -p public_html/uploads/profile_pictures
   chmod 755 public_html/uploads/profile_pictures

3. DATABASE SETUP
   ✓ Export your local database
   ✓ Import to Hostinger MySQL database
   (OR simply run the initialization script in browser)

4. VERIFY YOUR DATABASE
   In Hostinger control panel:
   ✓ Go to MySQL databases
   ✓ Go to phpMyAdmin
   ✓ Open your database
   ✓ Find 'users' table
   ✓ Check that profile_picture column exists
   ✓ If not, the system will create it automatically

5. UPDATE DATABASE CONNECTION (if needed)
   File: db_includes/db_connect.php
   
   Update these lines with your Hostinger credentials:
   - define('DB_HOST', 'your-hostinger-host');
   - define('DB_USER', 'your-hostinger-username');
   - define('DB_PASS', 'your-hostinger-password');
   - define('DB_NAME', 'your-hostinger-database');

6. TEST THE FEATURE
   ✓ Login to your application
   ✓ Go to My Profile
   ✓ Click Edit Profile
   ✓ Upload a test image
   ✓ Click Save Changes
   ✓ Verify image appears in profile

HOSTINGER-SPECIFIC NOTES:
========================

✓ File uploads: Hostinger supports file uploads (PHP handles it)
✓ Folder permissions: Use 755 for upload folders
✓ File size limit: Usually 128MB by default (our limit is 5MB)
✓ Storage: Plenty of storage available in all Hostinger plans
✓ Performance: File serving is optimized on Hostinger

TROUBLESHOOTING ON HOSTINGER:
=============================

If images don't upload:

1. Check folder permissions
   - Run: php admin_includes/init_profile_picture.php
   - This creates the folder with correct permissions

2. Check database connection
   - File: db_includes/db_connect.php
   - Verify credentials are correct

3. Check file permissions
   - Upload folder needs 755 permissions
   - Contact Hostinger support if stuck

4. Check PHP settings
   - Hostinger allows file uploads by default
   - Should work out of the box
